This repository contains implementations of the following models for multivariate time-series learning:

## Models

- **SaD**  
  *Structure-aware Decoupled Imputation Network for Multivariate Time Series*

- **HiCAT**  
  *Time-series Representation Learning via Hierarchical Contrasting and Augmentation*

## Requirements

- torch >= 1.10  
- numpy >= 1.20  
- pyyaml >= 6.0  
- tqdm >= 4.0
